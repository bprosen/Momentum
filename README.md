# Momentum
This plugin is made specifically for Renatus Network's Parkour server, it will not work correctly on other servers without specific conditions set.

<br />

**Key Features**<br />
- Checkpoints <br />
- Rankup <br />
- Prestiges <br />
- Plots <br />
- Races <br />
- Clans <br />
- Practice Mode <br />
- Infinite Parkour <br />
- Event System <br />
- Spectating <br />
- Detailed Statistics <br />
